Github Copliot 学生认证申请指北

发表于2026-03-18|更新于2026-03-18|生活指北
|浏览量:

Github Copilot 介绍

它是一款能在你编写代码时提供实时建议、自动补全代码，甚至根据注释生成代码片段的工具，旨在提高编程效率。
GitHub Copilot 提供了多个版本，包括免费版、Pro 版、Pro+ 版。

为什么申请学生认证

申请学生认证后可以免费使用两年Copilot Pro

详细教学

获取一个github账户

github官网

更改GitHub账号信息


邮箱认证

  • 你需要去学校的相关部门申请学校邮箱
       - 打开GitHub——点击头像——Settings——Emails，添加学校邮箱（.edu.cn），并进行验证，出现 Verified 即可：
        

支付信息填写


双因素身份认证

打开GitHub——点击头像——Settings——Password and authentication，添加双重身份认证，双重身份验证是在登录时要求不仅是密码这一项安全措施，还需要额外输入一个验证码，增添了一层额外的安全保障。

具体方法

安装 Microsoft Authenticator，打开之后扫描 GitHub 上的 QR码 进行账户绑定，输入软件提供的一次性密码，如下图表示验证成功：（出现 Configured 即可！）

更换定位

打开浏览器，推荐使用Google或Edge，按F12/Fn+F12打开开发者工具

  • Google用户：点击设置——Locations——Add locations，新增一个地址，填入你学校对应的经纬度以及地区和时间，之后保存退出即可

    Edge用户：点击传感器，没有的点旁边 +号找一下；然后点击管理新增一个你学校的定位

证明材料准备

以下证明材料选其一- 手写或电子版信息

  • 学生证
    确保所含内容务必包含：姓名、学校名称、学号、入学时间、毕业时间……
  • 录取通知书
  • 学校官网 (成功概率低)

GitHub Education认证

  1. 打开 settings 在这里点击开启认证即可
  2. 选择Student——直接点击Select this school（没有的话就在搜索框里搜索学校英文名字）——然后选择前面验证好的学校邮箱——f12选择学校的经纬度位置(这里位置错误的话后面会问你为什么不在学校)——之后点击Share Location（这里如果浏览器跳出来是否允许访问位置的话一定要点允许！！！）

  3. 选择你使用的材料对应的选项，如学生证选第一个
  4. 拍照或者上传你的材料
  5. 点击Submit Application
  6. 假如弹出询问为什么你的位置不在学校，就编写一个理由：
    • 出去打比赛
    • 没开学
    • 上网课
    • ……

申请成功，三天后即可使用
